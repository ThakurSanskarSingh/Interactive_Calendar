# Interactive Wall Calendar

A polished, responsive **wall calendar component** built with **Next.js, React, TypeScript, Tailwind CSS, Zustand, Framer Motion, and date-fns**.

**Live Demo:** [interactive-calendar-zeta.vercel.app](https://interactive-calendar-zeta.vercel.app/)

---

## Features

- **Interactive date range selection**
  - Select a single day or a full range
  - Hover preview while selecting
  - Handles reverse selection smoothly

- **Wall calendar layout**
  - Large seasonal hero image
  - Clean monthly date grid
  - Previous / next month navigation

- **Notes system**
  - Add notes for a selected day or date range
  - Sticky-note inspired notes UI
  - Notes saved in `localStorage`

- **Responsive design**
  - Side-by-side layout on desktop
  - Stacked layout on mobile
  - Touch-friendly interactions

- **Polished UX**
  - Smooth month transitions
  - Clear visual selection states
  - Accessible keyboard-friendly interactions

---

## Standout Features

- **Page flip month animation** for a real wall-calendar feel
- **Dynamic theme extraction** from hero images
- **Seasonal hero image rotation** based on month
- **Holiday markers** for special dates
- **Mini activity heatmap** showing note activity
- **Sticky-note style notes panel** for a more tactile experience
